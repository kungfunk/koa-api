# koa-api
Simple CRUD API REST Example using [koa](http://koajs.com/), [koa-route](https://github.com/koajs/route) and [monk](https://github.com/Automattic/monk) for learning purposes.

#### REST URIs
* GET /users
* POST /users
* GET /users/:username
* DELETE /users/:username
* PUT /users/:username
